INFO6019 Physics and Simulation (Project 3)

The purpose of this project is to learn and implement collision detection between particles and planes as well as between 2 particles.

CONTROLS
W/A/S/D	- Move the camera around the scene
Q/E	- Move the camera up and down in the scene
Mouse 	- Moving the mouse moves the camera's target allowing you to look around
1	- Launch a particle from the cannon
Space	- Launch a particle from the origin

Built and compiled using Visual Studios 2019 in Debug and Release in x64.
